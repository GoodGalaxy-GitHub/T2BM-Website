Getting Connected dns

Getting connected to T2BM is easy! You can use one of two methods to download and install the Better Minecraft pack to your game. While GDLauncher is the recommened method, you may also manually install the mod pack if you choose to. Keep in mind, however, that you will not receive automatic updates and you wil need to manually install any new versions of Better Minecraft that come out.

GDLauncher Method

Installing GDLauncher

GDLauncher is a very powerful client for Minecraft Java Edition. GDLauncher is able to natively support mods and can automatically install any mods from CurseForge. To install GDLauncher navigate to their website here. Choose the platform you're currently playing on and wait for the file to download. Once downloaded, simply open the file and follow the on-screen instructions.

Installing Better Minecraft

To install Better Minecraft, click the + button in the bottom left, select CurseForge and search for Better Minecraft. Make sure you choose the Fabric edition. Choose 'Download Latest' and click the arrow on the bottom right. Once the loading screen appears, click the X at the top right and wait for the mod files to download. Please keep an eye on the download! Some mod creators opt-out of third party launchers and you'll need to accept the manual download!


Manual Method

Installing Fabric

The first thing you'll need to do is install the Fabric loader. You can find the download at the official Fabric website here. I recommend that you select the universal .jar file, however you can use the Windows installer if you want. When you open the installer, make sure you're installing the client version, not the server version (see the tabs at the top). Choose the latest version of the loader and choose Minecraft 1.18.2. Make sure that your install location is correct and make sure create profile is checked. Click install and you're good to go!

Creating the profile

While this step is optional, I highly recommend you do it to make switching between the modpack and the vanilla version of the game easier. We'll be creating a profile that contains the modpack and you can choose this profile anytime you want to play T2BM. To do this, open the Minecraft Launcher, make sure you have Java Edition selected, and click the installations tab at the top. Click 'New Installation'. Give the installation a name and under version select 'release-fabric-loader-0.14.8-1.18.2'. Change the directory to something other than your .minecraft folder. I recommend creating a new one on your C drive.

Installing Better Minecraft

Now it's time to install Better Minecraft. Click here to download the latest version of the Better Minecraft mod pack. Extract the zip and move all the contents to the directory you chose during the profile creation.


Additional Setup

Using Voice Chat (opt.)

T2BM natively supports the Simple Voice Chat mod by henkelmax. Simple Voice Chat supports in-game chat groups and proximity chat.

If you're on GDLauncher

Right click the instance you created that contains Better Minecraft. Click manage and choose the mods tab. Click add a mod and search for 'Simple Voice Chat'. Find the 1.18.2 Fabric version of the mod and install it. Now you're ready to use voice chat! The default push-to-talk button is Caps Lock and you can perform additional configuration in-game.

If you're on the default launcher

Download the mod here. Drag and drop the .jar file into your mods folder where you installed your game, by default it should be in your .minecraft folder. The default push-to-talk button is Caps Lock and you can perform additional configuration in-game.
